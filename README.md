## 🛠️ The Mobile-First AI Workflow

These projects are brought to life through a highly optimized, on-the-go development stack:

* **Environment:** Mobile terminal environment (Termux/Ubuntu)
* **AI Agent:** **Claude Code** for autonomous scaffolding, styling, and rapid iteration
* **Core Tech:** HTML5, CSS3, vanilla JavaScript
* **Deployment:** GitHub Pages for instant, zero-config hosting
* **Design Philosophy:** Clean, responsive, and tailored for fast client feedback

By utilizing an AI-driven terminal workflow on mobile, I can iterate on client requests, tweak UI elements, and push live updates from anywhere.
